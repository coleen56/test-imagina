Sujet de test technique pour un contrat en alternance - Imagina Vannes
Enoncé : 
"Langage : php

Ci joint un fichier CSV représentant un champs avec des menhirs.

La taille du champs est de 10x10, il est composé de 2 types de case :

0 -> case vide

1 -> menhir

Le but est de calculer le nombre de cycles de propagation nécessaire pour parcourir toutes les cases vides. Un cycle se faisant en effectuant toutes les propagations sur les cases affectées.

Une action de propagation part d'une case d'origine et se propage vers les cases voisines hors diagonales. Nous partirons de la case (9,0) situé dans le coin inférieur gauche.

Ce test permettra surtout de comprendre votre manière d'approcher, d'analyser et d'exercer votre esprit logique sur un problème et ne se repose pas sur l'attente de la réponse adéquate.

Nous attendons de votre part, le code réalisé pour résoudre ce problème ainsi que votre réponse."
